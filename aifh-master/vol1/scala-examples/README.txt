AIFH Vol1  - Scala Code Examples
====
This folder contains the Scala examples for AIFH Volume 1. 

## Building the Examples
These examples are build using the Scala Build Tool (SBT).  The Scala build tool can be
found at the following URL.

http://www.scala-sbt.org/

To compile the examples, run the following command from the scala-examples directory.

```sbt clean package```

This will create a JAR file and place it in the following folder.

```.\target\scala-2.10\```
